COMPUTER

A computer is an electronic device that processes data, converting it into information that is useful to people

Input Devices

Input basically means getting data into the computer to be processed. Input devices includes Keyboard, mouse, trackball, touchpad, joystick etc.

Keyboard

A keyboard is a peripheral device that enables a user to input text into a computer or any other electronic machinery

Mouse

A mouse is a small device that a computer user pushes across a desk surface in order to point to a place on a display screen and to select one or more actions to take from that position.

RAM

(random access memory) is a computer's short-term memory, where the data that the processor is currently using is stored.

ROM

(Read-only memory) is a type of computer storage containing non-volatile, permanent data that, normally, can only be read, not written to. ROM contains the programming that allows a computer to start up or regenerate each time it is turned on.
