Creating Bootable Flash Drive

BACKGROUND INFORMATION FOR LEARNERS

At present, USB Drives has become popularized for their case of use and portability compare to Compact Disc (CD) and Digital Versatile Disc (DVD). USB drives are used to store, backup, and transfer data between computers

A bootable flash drive allows users to boot the computer from the file system using a flash drive rather than a hard drive. Using bootable flash drive lets you install operating systems such as Windows 7,8, and 10.

Commonly used program in creating Bootable USB flash drive

Rufus

- a utility used to to format and create bootable flash drives, such as USB keys or pen drives. Rufus is a free software program creates bootable USB drives. You can also install multiple operating systems on the same device to create multi-boot USB drives.

Power ISO

- is an image file processing tool that allows user to edit, open, extract, burn, create, compress, encrypt spit and convert ISO file.

installing Power ISO is a one way of creating a bootable flash drive and burning on a disc. An ISO image also called a disc image is a file copy of an entire data CD or DVD. When you bum a CD or DVD from an ISO file, the new disc has the same folders, files and properties as the original disc. One way to get an ISO file is to download it from a website

CREATE A BOOTABLE USB DRIVE WITH RUFUS

STEP 1. Open up Rufus and plug your clean USB stick into your computer.

STEP2. Rufus will automatically detect your USB. Click on Device and choose the USB you want to use from the drop-down menu.

STEP 3. Make sure the Boot Selection option is set to Disk or ISO image then click Select. Rufus will open a file explorer browser window, locate the ISO image you want to burn onto the USB and select it.

STEP 4. Keep the default settings as they are. However, you can also change these if you want.

STEP 5. Type the Volume label a name of your choice, or leave it as it is, and click Star.

STEP 6: You will get a warning saying that any data on the USB will be destroy (this is normal). Click OK and Rufus will start creating the IS0 image on the USB.

STEP 7: Once Rufus has finished creating the IS0 image on the USB, click close.

INSTALLING POWER ISO IMAGE BURNER ON YOUR COMPUTER

STEP 1. Locate your Power ISO file and Open the folder

STEP 2. Select the Power ISO setup

STEP 3. Right-click on the Power ISO setup and click ""Run as Administrator" then click Yes.

STEP 4. Click Install

STEP 5. When the Installation Complete Click Next and Close.

PREPARING BOOTABLE USB FLASH DRIVE USING THE POWER ISO

STEP 1. Look for the Power ISO icon, right click and select "Run as Administrator"

Note: if there is no icon on the desktop, you can use the search windows to locate the software.

STEP 2. Click on the "Tools tab"` and select "create bootable USB Drive"

STEP 3. Click the folder icon on the source image file.

STEP 4. Locate the location of the Windows 7 ISO file and click "Open"

STEP 5. Select the correct USB drive from the "Destination USB Drive" list if there are two or more USB drives connected on the computer

STEP 6. On the Setting, select USB-HDD write method and click on start button.

STEP 7. Do not abort or cancel the process, the process will finish within 10 to 20 minutes depending on the speed of the computer. When the process completed just click close and eject your newly created bootable USB drive.
